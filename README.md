## Описание
Мобильное приложение **Bird** 
разработано в рамках учебной практики. 
Приложение позволяет пользователю вести дневник важных задач и получать уведомления для своевременного выполнения заданий.
В приложении реализован функционал записи к врачу по симптомам болезни, доступный через боковое меню. 
Процесс включает ввод симптомов, подбор врача на основе указанных симптомов, выбор даты и времени приема. 
После подтверждения записи она автоматически добавляется на главный экран в виде отдельной задачи.

Приложение отсчитывает время до дедлайна каждой задачи и отправляет уведомления за установленный пользователем промежуток: 15, 10 или 30 минут.

## Установка
Для установки программы:
1. Зайдите в раздел `releases`
2. Установите `.apk` из самого нового релиза
3. Найдите скачанный файл в файловом менеджере
4. Нажмите на файл и подтвердите установку
5. После завершения установки приложение будет доступно на устройстве

## Использование
Мобильное приложение **Bird** создано для предупреждения «пожаров» дедлайнов. Приложение будет напоминать о важных задачах, как птичка, принося важную весточку:
- О домашней работе
- О сроках сдачи
- Об оставшемся времени

**Возможности:**
- Планирование графика дня/недели/месяца
- Отслеживание стадии выполнения задач
- Отчеты о выполнении заданий
